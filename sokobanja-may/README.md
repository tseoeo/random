# sokobanja-may

Mobile-friendly single-page trip notebook for a 3-night trip to Sokobanja, Serbia (22–25 May 2026).
Hiking + sights + flora/fauna only — no cycling content (the bikes stayed home this trip).

- `index.html` — the page.
- `style.css` — limestone-karst palette, mobile-first, no framework.
- `script.js` — Leaflet + leaflet-gpx initialiser; OSM tiles; no API key.
- `gpx/` — five hike GPX tracks plus one optional longer Ozren traverse, sourced from PSS / Planine.net / Komoot / Sokobanja recreational map.

Open `index.html` directly in a browser. Maps need an internet connection (Leaflet + tiles are loaded from CDN); the rest of the page works offline.
